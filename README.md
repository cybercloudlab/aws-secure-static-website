# Secure Static Website Hosting on AWS

## Project Overview
Hands-on AWS project demonstrating secure static website hosting with a focus on cloud security best practices.

## AWS Services Used
- S3 (Static Website Hosting)
- CloudFront (CDN)
- IAM (Access Control)
- Optional: CloudTrail & GuardDuty (Security Monitoring)

## Folder Structure
- `architecture/` → diagrams and architecture visuals
- `screenshots/` → screenshots of AWS console/configuration
- `notes/` → explanations of security decisions
- `lessons-learned.md` → personal learning points
